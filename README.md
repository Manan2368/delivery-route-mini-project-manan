# Delivery Route Optimization

This project implements multiple algorithms:
- Recursion (TSP)
- Greedy (Parcel Selection)
- Dynamic Programming (Knapsack)
- Graph Algorithms (Dijkstra, MST)

## How to Run
pip install -r requirements.txt
jupyter notebook

## Output
- Optimized delivery route
- Parcel selection
- Performance graphs
